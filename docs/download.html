PyLangAcq is compatible with both Python 2 and 3 (specifically, 2.7 and 3.4+). It is available via pip:

$ pip install pylangacq

The source code for development is hosted on GitHub.

To test your installation:

>>> import pylangacq
>>> pylangacq.__version__  # show version number

The Tutorial page introduces the basic functionalities of PyLangAcq.
