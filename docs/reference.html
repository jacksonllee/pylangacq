Library reference¶

pylangacq.read_chat(*filenames, **kwargs)[source]¶

Create a Reader object based on filenames.

Parameters:
  • filenames – one or multiple filenames (absolute-path or relative to the current directory; with or without glob matching patterns)
  • kwargs – Only the keyword encoding is recognized, which defaults to ‘utf8’. (New in version 0.9)
